# Nvidia-Fibernet Broadband Management System (Java)

The Broadband Management System is a desktop-based application developed using Core Java and MySQL to manage broadband customer records and billing operations.

## Features
- Add, update, delete customer details
- Manage broadband plans and billing information
- Perform CRUD operations using JDBC
- Store and retrieve data from MySQL database
- Simple and user-friendly interface

## Technologies Used
- Core Java (Java 8)
- JDBC
- MySQL
- OOP Concepts
- Exception Handling

## Functional Modules
- Customer Management
- Billing Management
- Database Integration

## Learning Outcome
- Gained hands-on experience in database connectivity using JDBC
- Strengthened understanding of CRUD operations
- Improved knowledge of backend system design
